Guide for the Simon Says Game

  • Press any of keyboard key to start thr game
  • There is total 10 levels to won this game
  • In each level one of the button is flashing
  • To pass the level user has to enter the correct path from the flashed button in sequence
  • Ex: If lvl1:green and in lvl2:red then corrct sequence = "green" , "red"
  • Each of level consist the 1point
  • Any wrong guess in sequence turns into "GAME OVER"
